The NVIDIA Stack
It runs entirely on the DGX Spark.
Local inference via NVIDIA NIM. Llama 3.1 8B served on an OpenAI-compatible endpoint — no cloud round-trip.
Private by design. A buyer’s address and finances never leave the device. The whole pipeline is on-box.
Headroom to scale. A config toggle swaps in a larger NIM model when the moment calls for more reasoning.
